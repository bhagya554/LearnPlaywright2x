# LearnPlaywright2x

A learning repository for Playwright automation testing with JavaScript.

## Contents

- **Chapter 1: Basics** - JavaScript setup, commands, and first programs
- **Chapter 2: JavaScript Concepts** - Core JS language concepts
- **Chapter 3: Identifiers & Literals** - Identifier rules, naming conventions, comments
- **Chapter 4: JavaScript Concepts** - Variables (`var`, `let`, `const`), functions, scoping, and hoisting
- **VSCode Shortcuts** - Keyboard shortcut references for Windows and macOS

## Prerequisites

- Node.js
- Visual Studio Code
- Playwright
