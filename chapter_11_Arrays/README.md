# Chapter 11: Arrays

Creating, accessing, modifying, searching, iterating over, and transforming JavaScript arrays.

## Files

| File | Topic |
|------|-------|
| `83_Arrays.js` | Array basics — literal syntax, indexing, mixed-type arrays, `undefined` for out-of-range index |
| `84_Arrays.js` | Array creation — `Array` constructor, `Array.of`, `Array.from` |
| `85_Access_Array.js` | Accessing elements with `[]` and `at()` (negative indexes), modifying by index |
| `86_Array_Adding_Remove.js` | `push` / `pop` / `unshift` / `shift` — add and remove from either end |
| `87_Adding_Remove2.js` | `splice` — insert, remove, and replace elements at any position |
| `88_Real_Example.js` | Realistic example — browser list, `length`, removing and iterating |
| `89_Searching.js` | Searching — `indexOf`, `lastIndexOf`, `includes`, `find`, `findIndex`, `findLast`, `findLastIndex` |
| `90_IterateOverArray.js` | Iteration — classic `for`, `for...of`, `forEach`, `for...in` |
| `91_Transform_Array.js` | Transformation — `map`, `filter`, `reduce` (with accumulator details), `flat` |
| `92_Arrays.js` | Sorting — `sort()` default lexicographic vs numeric compare function `(a, b) => a - b` |
| `93_Array_Slicing.js` | `slice` vs `splice` — non-mutating slice (with negative indexes) vs mutating splice |
| `94_Concat_Array.js` | Combining arrays — `concat`, spread `[...a, ...b]`, and `join` |
| `95_Array_Checking.js` | Validation — `Array.isArray`, `every`, `some` (with a Playwright status-code example) |
